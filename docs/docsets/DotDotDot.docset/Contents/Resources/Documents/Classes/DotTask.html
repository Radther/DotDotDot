DotTask

A DotTask (or simply task) instance is an object that can use a DotRequest. This is done by calling the start method. A task is reusable. Start can be called as many times as you like. This means if a request needs to be repeated (such as pull-to-refresh) the task can be started again. If a current task is already running this will be canceled and a new one will be started.
